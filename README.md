# Custom ReactJS-Webpack Boilerplate

This repo include configuration for:

- ReactJS
- Webpack 5
- ESLint/Airbnb (Style Guide)
- Prettier (code format)
- Sass
- Babel
- Husky (improves your commits and more woof!)

### Installation

```
$ yarn install
```

### Scripts

```
// Development mode
$ yarn start

// Build app
$ yarn build

// Format with prettier
$ yarn format

// Lint
$ yarn lint --fix
```
